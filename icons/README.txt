Petunjuk Manual Penambahan Ikon PWA:

Untuk melengkapi aplikasi PWA Anda, silakan buat dua file gambar dan letakkan di dalam folder 'icons' ini.

1. Nama file: icon-192x192.png
   Ukuran: 192 x 192 piksel

2. Nama file: icon-512x512.png
   Ukuran: 512 x 512 piksel

Pastikan nama file dan ukurannya sesuai agar aplikasi dapat diinstal dengan benar di perangkat pengguna.
